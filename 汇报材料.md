# 🎭 戏韵万象——京剧数据可视分析系统

## 一、项目概况

本系统是一个针对京剧剧本的深度可视化分析平台，基于 **1,473 部京剧剧本**、**26,877 个角色**的数据，结合自然语言处理（TF-IDF、PCA）、大语言模型（Qwen2.5-7B）以及复杂网络分析方法，从**行当推断、角色关系网络、主题建模、叙事结构**四个维度对京剧进行全方位的量化分析，并在第五个任务中整合四个维度进行跨维度关联分析。

**项目地址：** https://github.com/huangyapingde/China_vis-all-data-
**在线演示：** https://huangyapingde.github.io/China_vis-all-data-/

## 三、分析任务与核心发现

### 任务一：行当推断与分析

**分析方法：** 提取角色台词长度、频次、情感词频、权力指令频次等行为特征，利用 PMI（点互信息）、纯度和熵度量角色类型与生、旦、净、丑四大行当的关联度，利用大模型推断未标注角色的行当。

**可视化图表（5张）：**
1. 角色类型→行当映射桑基图
2. 行当-角色 PMI 关联强度 TOP15
3. 各行当纯度与熵
4. 角色特征聚类散点图
5. 各时期行当占比变化

**核心发现：**
- 武将、奸臣与"净"行（花脸）绑定极强；将军、官员与"生"行高度吻合；后妃、仆役、女将几乎全部归于"旦"行
- 行当并非随意分配，而是与角色的社会身份、性格特征有着近乎固定的对应关系——京剧的行当体系本质上是一套"角色类型化"的编码系统
- 四大行当比例在各卷之间保持了惊人的稳定性——生行约占 40%，旦、净、丑各约占 20%，说明行当结构是京剧作为一种艺术形式的"深层语法"

---

### 任务二：角色关系网络分析

**分析方法：** 构建共现层（场景共现）、对话层（台词承接）和权力层（命令与服从）三层网络，基于 PageRank、度中心性、中介中心性等多维指标评估角色影响力，利用 Louvain 算法识别阵营社区。

**可视化图表（4张）：**
1. 剧种网络密度对比
2. 角色关系网络图（力导向布局，可切换剧种）
3. 角色影响力雷达图
4. 角色中心性排名（跨剧本聚合）

**核心发现：**
- **历史戏**呈"众星捧月"式的星型结构——一个核心角色主导，权力网络密度显著高于其他剧种
- **家庭戏**呈多中心分散结构——角色间情感互动紧密但权力层级低
- **公案戏**呈官员-平民-罪犯三方对峙结构——对话网络最密集，台词交锋活跃
- 诸葛亮、关羽、包拯等传统大角色在多个维度上同时突出，是京剧世界中最核心的角色

---

### 任务三：主题建模与分析

**分析方法：** 基于 15 类主题关键词词典进行关键词匹配与 TF-IDF 向量化，使用 PCA 投影探索主题空间，计算主题共现 PMI 与关联规则，并利用 Qwen2.5:7b 进行大模型标注验证。

**可视化图表（4张）：**
1. 剧种×主题强度对比
2. 主题空间分布（PCA 降维）
3. 主题共现网络
4. 主题随场景演化曲线

**核心发现：**
- **历史戏**呈现"战争军事+智谋策略"的二元主题结构
- **家庭戏**以"家庭伦理+爱情情感"为单核，悲情苦难成分显著
- **公案戏**是"正义审判+权力斗争+悲情苦难"的多主题复合体
- "战争+智谋"、"忠诚+君臣"、"家庭+爱情"是京剧最稳定的三类主题搭配组合

---

### 任务四：叙事结构分析

**分析方法：** 综合情绪张力、冲突强度和事件密度特征，利用高斯滑动窗口平滑与 SciPy 峰值检测识别叙事高潮，进行"起、承、转、合"四阶段自动分割，识别单峰/双峰/多峰叙事模式。

**可视化图表（4张）：**
1. 不同剧种叙事模板曲线
2. 叙事结构类型分布
3. 高潮位置分布
4. 叙事曲线图（四线叠加，可交互选择剧本）

**核心发现：**
- 约 **87%** 的京剧剧本采用单峰结构——一个核心高潮前后展开，符合"起承转合"的经典叙事逻辑
- 历史戏呈现经典的"渐进式上升"曲线，高潮位于中后段（约 65% 位置）
- 家庭戏曲线波动最为剧烈，多峰结构占比最高
- 公案戏呈现"调查→冲突→判决"的三段式节奏

---

### 任务五：跨维度关联分析

**分析方法：** 将网络、主题、叙事三个维度的 16 个指标对齐到 878 部全维度剧本，计算 Pearson 相关系数矩阵，通过规则挖掘发现典型关联模式，用 K-means 聚类识别四类典型结构。

**可视化图表（3张）：**
1. 跨维度关联矩阵（16×16 热力图）
2. 平行坐标探索（9 维联合观察）
3. 典型关联模式与结构类型（含雷达对比）

**核心发现：**

| 模式 | 关联规律 | 支持度 |
|------|---------|--------|
| P1 | 高模块度 → 主题多样化 | 100% |
| P2 | 高中心化 → 单峰线性叙事 | 92% |
| P3 | 高权力密度 → 高潮前置 | 28% |
| P4 | 多角色低模块度 → 群像交织叙事 | 9% |
| P5 | 张力波动大 → 多峰结构 | 14% |
| P6 | 主题多样 → 高潮靠后 | 34% |

**四类典型结构：**
- **权力集中型**（73 部）：高中心化、高权力密度、单峰叙事
- **群像交织型**（42 部）：多角色、低模块度、主题多样
- **情感驱动型**：少角色、高聚类系数、多峰结构
- **传统叙事型**：中等各项指标、单峰标准结构

## 五、关键结论（汇报要点）

1. **行当体系是京剧的角色编码系统**——角色类型与行当之间存在近乎固定的对应关系，且这种结构在历史上保持稳定
2. **不同剧种的"网络指纹"不同**——历史戏呈星型、家庭戏呈分散型、公案戏呈对峙型
3. **每类剧种有自己的"主题配方"**——历史戏=战争+智谋，家庭戏=伦理+情感，公案戏=正义+权力
4. **87% 的京剧采用单峰叙事**——符合"起承转合"的经典结构，但不同剧种的节奏曲线各有特征
5. **三维度之间存在系统性耦合**——"人物配置—主题选择—节奏安排"三者存在可量化的关联规律
